#  Book-Shop - Интернет-магазин книг

> Полноценное SPA-приложение интернет-магазина книг, разработанное на чистом HTML, CSS и JavaScript.

##  О проекте

Этот проект создан для демонстрации навыков фронтенд-разработки. Он включает все основные сценарии пользователя: от просмотра каталога до оформления заказа. Проект полностью работает в браузере без бэкенда - все данные хранятся в `localStorage`.

---

##  Функционал

| Модуль | Описание |
|--------|----------|
| **Каталог книг** | Отображение книг с ценами, скидками (до 50%), фильтрация по категориям |
| **Корзина** | Добавление/удаление, изменение количества, авторасчет доставки (бесплатно от 1500 ₽) |
| **Избранное** | Сохранение книг в отдельный список с возможностью добавить все в корзину |
| **Поиск** | Интерактивный поиск с автоподсказками по названию и автору |
| **Личный кабинет** | Регистрация, авторизация, редактирование профиля, смена аватара, история заказов |
| **Оформление заказа** | Форма с адресом доставки, выбором способа оплаты, расчет итоговой суммы |
| **Адаптивность** | Корректное отображение на мобильных устройствах, планшетах и десктопах |

##  Стек технологий

| Технология | Что используется |
|------------|------------------|
| **HTML5** | Семантическая разметка, формы, модальные окна |
| **CSS3** | Flexbox, Grid, адаптивная верстка (Mobile First), анимации (`@keyframes`), кастомные слайдеры |
| **JavaScript (ES6+)** | DOM, события, асинхронность (`fetch`, `async/await`), работа с `localStorage`, валидация форм |
| **Figma** | Дизайн-макеты, верстка по ним с соблюдением дизайн-системы |
| **Boxicons** | Иконки для интерфейса |

## Известные проблемы и планы по исправлению

> Я знаю о них и планирую исправить в ближайшее время:

| Проблема | Описание | Решение |
|----------|----------|-------|
|**Загрузка изображений** |	Некоторые обложки книг могут не отображаться, если путь к файлу указан неверно |	Проверить все пути в `data.js` и привести к единому формату |
|**Адаптивность главного слайдера** |	На очень маленьких экранах (менее 320px) текст слайдера выходит за границы |	Добавить медиа-запросы для экранов < 320px |
|**Валидация формы регистрации** |	При вводе некорректного email ошибка показывается, но поле не подсвечивается |	Добавить визуальную индикацию ошибок (красная рамка, иконка) |
|**Корзина на мобильных** |	На узких экранах кнопка «Оформить заказ» может перекрываться содержимым |	Уменьшить отступы и `padding` в мобильной версии |
|**Поиск** |	При пустом запросе показывается уведомление, но фокус остаётся в поле ввода |	Автоматически очищать поле или убирать фокус после поиска |
|**Карточки товаров** |	При наведении на карточку появляется оверлей, но на тач-устройствах он не исчезает |	Добавить обработку `touch-событий` для мобильных устройств |
|**Сохранение состояния** |	При перезагрузке страницы активная вкладка в личном кабинете сбрасывается |	Сохранять активную вкладку в `localStorage` |
|**Доступность (a11y)** |	Не хватает aria- атрибутов для людей с ограниченными возможностями |	Добавить `aria-label, role` и другие атрибуты |
